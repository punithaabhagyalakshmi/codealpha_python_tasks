# codealpha_python_tasks
# Basic Chatbot - CodeAlpha Task 4

# Description
This is a simple rule-based chatbot built using Python.  
The chatbot responds to basic user inputs such as greetings and farewell messages using predefined replies.  
It runs in the console and continues chatting until the user types "bye".

# Features
- Responds to "hello"
- Responds to "how are you"
- Ends chat on "bye"
- Handles unknown inputs
- Runs in continuous loop until exit

# Concepts Used
- if-elif-else conditions
- Functions
- Loops
- String handling
- Console Input/Output
